## Installation
### 1) Up docker
``$ make build``<br> 
``$ make up``
### 2) Enter to app
``$ make enter``
### 3) Migrate database
``$ bin/console doctrine:migrations:migrate``
### 4) Create user
### For admin
``$ bin/console app:create:admin <email> <password>``
### For default user
``$ bin/console app:create:user <email> <password>``
